### Local Development

To run locally:

```bash
# Option 1: Using Python
python3 -m http.server 8000

# Option 2: Using Node.js (if you have npx)
npx serve .

# Option 3: Using any other static file server
```

Then open `http://localhost:8000` in your browser.
